\subsection*{Comentario inicial}
En Java, a diferencia de C, un mismo proyecto puede contener más de una función \texttt{main()}. Más concretamente, cada clase puede tener una función \texttt{main()}, por eso es necesario indicar qué clase se quiere empezar a ejecutar cuando se invoca la máquina virtual de Java. En las prácticas puede integrar la función \texttt{main()} dentro de la clase (y por lo tanto el mismo archivo), o puede crear un archivo nuevo dentro del mismo paquete y ubicar ahí el \texttt{main()}. Se sugiere integrar el \texttt{main()} dentro de la clase, dado que es la práctica más habitual.

\subsection*{Ejercicio 1}
Implemente la clase Coche que represente coches. Incluya los atributos marca, modelo, marcha y color; y los métodos que simulen las acciones de arrancar el motor, describir el coche, cambiar de velocidad (subir y bajar marcha), acelerar, frenar y parar el motor (simplemente muestre un texto por pantalla o modificando algún atributo).\\

Muestre el funcionamiento de la clase creando tres instancias de la clase representando coches diferentes, llamando a sus métodos y visualizando el resultado por pantalla.

\subsection*{Ejercicio 2}
Cree una clase Polinomio que implemente el siguiente polinomio:

\begin{equation}
f(x) = a x^3 + b x^2 + c x + d
\end{equation}

Con los atributos \texttt{a}, \texttt{b}, \texttt{c} y \texttt{d}, de tipo \texttt{double}, y los métodos siguientes:

\begin{enumerate}
\item \texttt{public void setA(double a)}: Establece el valor del coeficiente a.
\item \texttt{public void setB(double b)}: Establece el valor del coeficiente b.
\item \texttt{public void setC(double c)}: Establece el valor del coeficiente c.
\item \texttt{public void setD(double d)}: Establece el valor del coeficiente d.
\item \texttt{public void setABCD(double a, double b, double c, double d)}: Establece el valor de los coeficientes a, b, c y d.
\item \texttt{public double compute(double x)}: Calcular el valor del polinomio para la x dada.
\end{enumerate}

\subsection*{Ejercicio 3}
Diseñe e implemente una clase Calculadora que calcule la suma, resta, división y multiplicación de dos números que se pasarán como argumento a los métodos. ¿Es necesario utilizar atributos? Discútalo con el profesor del laboratorio.

\subsection*{Ejercicio 4}
Implemente una clase TableroAjedrez que visualice en modo texto un tablero de ajedrez (8x8), usando 'N' para identificar casillas negras, y 'B' las casillas blancas. Implementar un método adicional que imprima el tablero, pero indicando además a qué casillas se puede mover un alfil dada una determinada posición en forma (fila, columna).

Utilice los siguientes métodos:

\begin{enumerate}
\item \texttt{pubic void imprimirTablero()}: Imprime un tablero sin fichas.
\item \texttt{public void imprimirTablero(int fila, int columna)}: Imprime el tablero y las posiciones a las que se puede mover un alfil en la ubicación dada.
\end{enumerate}

\textit{Optativo: Diseñe (e implemente) una nueva clase que permita visualizar el tablero y los movimientos válidos de varias figuras (por ejemplo, torre, caballo y alfil) atendiendo a los principios de modularidad, encapsulamiento y reutilización. Discuta el diseño con el profesor del laboratorio.}

\subsection*{Ejercicio 5}

\subsection*{Ejercicio 6}
Hacer un proyecto que integre el cálculo de polinomio del ejercicio 2 y la calculadora del ejercicio 5, de tal manera que todos los cálculos en la clase Polinomio se realicen invocando a la calculadora. El proyecto debería estar dividido en las siguientes clases con sus correspondientes paquetes:

\begin{enumerate}
\item \texttt{p2.ej6.calculadora.InterfazCalculadora}: Interfaz que define las operaciones de la calculadora.
\item \texttt{p2.ej6.calculadora.Calculadora}: Clase que implementa la interfaz anterior.
\item \texttt{p2.ej6.polinomio.Polinomio}: Clase desarrollada en el ejercicio 2 modificado para usar la clase \texttt{Calculadora}.
\item \texttt{p2.ej6.Main}: Clase que contiene la función \texttt{main()}.
\end{enumerate}
